# 🛡️ Military Asset Management System (Frontend)

A modern and responsive Military Asset Management System built with **React.js**, **Vite**, **Tailwind CSS**, and **Axios**. The application provides an intuitive dashboard for managing military assets, tracking assignments, monitoring expenditures, and maintaining secure asset records through REST API integration with the backend.

---

## 📌 Project Overview

The Military Asset Management System is designed to simplify the management of military equipment and resources by providing a centralized platform for administrators. It allows users to efficiently maintain asset records, monitor asset assignments, and generate reports through a clean and responsive interface.

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/Rushikesh1Avachat/frontend-military-assest-management-system.git

cd frontend-military-assest-management-system
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env` file in the project root.

```env
VITE_API_BASE_URL=http://localhost:5000/api
```

---

## ▶️ Run Development Server

```bash
npm run dev
```

Application will run at

```
http://localhost:5173
```

---

## 🏗️ Build Production

```bash
npm run build
```
